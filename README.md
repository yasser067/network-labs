# Network Labs

This repository contains networking labs I built while studying Network and Security at EST.

## Topics Covered
- VLANs
- Trunking
- DHCP
- Static Routing
- OSPF
- ACLs

## Labs

### VLAN Lab
- Created VLAN 10 and VLAN 20
- Configured access ports
- Configured trunk links
- Verified connectivity

### DHCP Lab
- Configured DHCP pools
- Tested automatic IP assignment
